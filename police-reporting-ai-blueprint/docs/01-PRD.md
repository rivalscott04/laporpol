# PRD

## Roles
- Super Admin
- Admin
- User

## Admin
- CRUD User
- CRUD Report
- Search & Filter
- Daily/Weekly/Monthly Recap
- Export Excel/PDF
- Audit Log

## User
- Edit Profile
- CRUD Report

## Report Fields
- Date
- Latitude
- Longitude
- Location Name
- Photo (server watermark)
- Attachment PDF/JPG
- Notes

Business Rules:
- Latitude/Longitude readonly from browser geolocation.
- Timestamp watermark generated by server.
- Soft deletes.
